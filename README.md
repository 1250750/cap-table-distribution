# Cap Table Distribution

Interface simples e editavel para visualizar uma cap table com equity vested agora, equity futura, pools transparentes e vesting linear por data de inicio.

## Funcionalidades

- Pie/donut chart com modo "Vested agora" e "Futuro total".
- Editor de nomes, tipos, percentagens, datas de inicio e anos de vesting.
- Pools mais transparentes para distinguir reservas futuras.
- Guardado localmente no browser via `localStorage`.
- Calculo automatico de equity vested, unvested, por alocar e acima de 100%.

## Desenvolvimento

```bash
npm install
npm run build
```
